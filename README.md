# Horno de Momos
Un instalador de mods bastante rudimentario para Super Momos Crushers escrito en Batch
## Uso
> [!NOTE]
> Siempre haz una copia de seguridad de los archivos antes de continuar, tampoco utilices mods que sean de otras versiones

Simplemente indica la direccion del mod y de tu juego, asi de simple
### Borrar un mod
1. Entra en la carpeta de tu juego
2. Elimina la carpeta `www`
3. Renombra `www_vanilla` a `www`
